# Quick Start Guide

## Get Started in 3 Steps

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Application
```bash
python app.py
```

### 3. Open Your Browser
Navigate to: **http://127.0.0.1:5000**

---

## What You Can Do

### 📝 Add Practice Questions
- Go to **Questions** → Click **"Add Question"**
- Fill in details: title, category, difficulty, solution
- Track coding, system design, behavioral, and SQL questions

### 📊 Track Job Applications
- Go to **Applications** → Click **"Add Application"**
- Monitor company, role, stage, and feedback
- Keep all your interviews organized

### 📚 Organize Study Materials
- Add markdown files to `material/` folder
- Organize by topic (data structures, algorithms, system design)
- Reference articles and summaries

### ✅ Monitor Progress
- Use the **Syllabus** page to track topics
- Mark items as completed
- Focus on high-priority areas

### 🏢 Research Companies
- Add interview process info to **Company Styles**
- Document rounds, tips, and common questions
- Add company-specific notes

### 🔗 Save Resources
- Bookmark useful links in **Links** section
- Organize by category
- Quick access to learning resources

---

## Tips for Success

1. **Daily Practice**: Add at least one question per day
2. **Document Everything**: Write down interview feedback immediately
3. **Track Progress**: Update syllabus status regularly
4. **Research Companies**: Add company info before interviews
5. **Use Sample Content**: Check the example files for formatting ideas
