# Interview Questions

This directory stores practice interview questions and answers organized by topic and difficulty.

## Structure
- Organize by category (coding, system design, behavioral, SQL, etc.)
- Each question can be a markdown file or stored in the database
- Include solution approaches, time/space complexity, and key insights

## Categories
- **Coding**: Algorithm and data structure problems
- **System Design**: Architecture and design problems
- **Behavioral**: STAR method responses
- **SQL**: Database queries and optimization
- **Data Engineering**: Pipeline design, ETL, data modeling

## Format Example
```markdown
# Question Title

**Difficulty**: Easy/Medium/Hard
**Company**: Google, Meta, etc.
**Topic**: Arrays, Trees, etc.

## Problem Statement
[Description of the problem]

## Solution Approach
[Your approach and reasoning]

## Code
```python
# Your solution
```

## Complexity
- Time: O(n)
- Space: O(1)

## Key Insights
[Important takeaways]
```
