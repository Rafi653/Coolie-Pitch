# Study Material

This directory contains reference articles, study summaries, and learning resources for interview preparation.

## Structure
- Organize by topic (e.g., `data-structures/`, `system-design/`, `algorithms/`)
- Use markdown files for notes and summaries
- Can be manually created or scraped from web resources

## Example Topics
- Data Structures (Arrays, Trees, Graphs, Hash Tables, etc.)
- Algorithms (Sorting, Searching, Dynamic Programming, etc.)
- System Design concepts
- Database fundamentals
- Distributed systems
- Big Data technologies (Spark, Kafka, etc.)

## Adding Material
Create markdown files with clear titles and organize by topic:
```
material/
  data-structures/
    arrays.md
    linked-lists.md
  system-design/
    scalability.md
    caching.md
```
