# Company Interview Styles

Document interview processes and styles for different companies.

## Structure
- One file or entry per company
- Include rounds, typical questions, tips, and resources
- Can be scraped from Glassdoor, Blind, LeetCode, etc.

## Information to Capture
- Number of interview rounds
- Types of interviews (coding, system design, behavioral, cultural fit)
- Duration of each round
- Difficulty level
- Common topics
- Company-specific tips
- Timeline from application to offer

## Example: Google
```markdown
# Google Interview Process

## Overview
- Typically 5-6 rounds
- Strong focus on algorithms and data structures
- Behavioral questions based on "Googleyness"

## Rounds
1. **Phone Screen** (45 min)
   - 1-2 coding questions
   - Medium difficulty
   
2. **Onsite** (4-5 rounds, 45 min each)
   - Coding (2-3 rounds)
   - System Design (1 round)
   - Behavioral/Googleyness (1 round)

## Tips
- Focus on optimal solutions
- Communicate your thought process
- Ask clarifying questions
- Consider edge cases

## Resources
- [Google Interview Guide](example.com)
```
